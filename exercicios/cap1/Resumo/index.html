Tags em HTML5

Em HTML5, podemos inserir os elementos da seguinte lista. Passe o mouse sobre os termos para descobrir qual tag utilizar em cada caso.

  • Título
  • Parágrafo
  • Imagens
  • Abrviações
  • Códigos-fonte
  • Listas numeradas
  • Listas com demarcadores
  • Quebra de linha

Para semântica temos:

  • Negrito
  • Itálico
  • Texto marcado
  • Sublinhado
  • --Subescrito--
  • --Sobrescrito--
  • Citações simples
  • Citações completas
  • Texto grande
  • Texto pequeno
  • Texto deletado
  • Texto inserido
  • Texto invertido

Mídia

É muito importante utilizar a tag picture e source midia type para adaptarmos as imagens e coisas do tipo para cada tipo de dispositivo, baseando-se no tamanho de pixels que o dispositivo comporta.

Já para áudios, basta utilizar a tag audio somente, ou inserir a tag source type junto.
